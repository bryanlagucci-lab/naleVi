# NaleVI — Professional Portfolio
**Nabukeera Leah Victoria (NALEVI)**

A premium, multi-page professional portfolio website designed for a dynamic Ugandan leader and creative professional. This project showcases a multifaceted career spanning writing, business coaching, administration, fashion styling, and advocacy.

---

## 👨‍💻 Developer
**Bryan Dev**

## ✨ Core Features
- **Modern & Premium Aesthetic**: Sophisticated design palette featuring glassmorphism, smooth gradients, and vibrant accents.
- **Interactive Experience**: 
  - **Dynamic Articles Slider**: Semi-transparent Mosaic captions with hover transitions.
  - **Custom Animated Cursor**: A premium desktop navigation experience.
  - **Scroll Reveal Animations**: Elements fade and slide into view for a lively user journey.
- **Multifaceted Content**:
  - Detailed **6-Role Breakdown** with specialized sidebar information.
  - **Interactive Skills Bars** visualizing core competencies.
  - **Testimonials & Success Stories** from diverse industry collaborations.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewports.
- **Performance Optimized**: Built with lightweight, library-free HTML, CSS, and Vanilla JavaScript.

## 🛠️ Technology Stack
- **Frontend Core**: HTML5, Vanilla CSS3 (Custom Design System).
- **Interactivity**: Vanilla JavaScript (ES6+).
- **Typography**: Playfair Display (Serif) & DM Sans (Sans-serif) via Google Fonts.
- **Icons**: Custom-curated professional iconography.

## 📂 Project Structure
- `index.html`: Professional home page with marquee and impact stats.
- `about.html`: In-depth biography and personal values.
- `roles.html`: Comprehensive breakdown of professional services.
- `skills.html`: Detailed visualization of expertise and tools.
- `testimonials.html`: Client feedback and collaboration history.
- `contact.html`: Multichannel engagement and enquiry forms.
- `styles.css`: Centralized design system and responsive styles.
- `script.js`: Global animations and custom interactions.

## 🚀 Development Philosophy
This project was built focusing on **Performance, Elegance, and Simplicity**. By avoiding heavy external libraries, the site maintains rapid load times while providing a state-of-the-art interactive experience that effectively communicatesprofessional excellence.
